Careers

We are seeking for skilled professionals:


Oracle PL/SQL Developer

  • At least 1 year experience using Oracle PL/SQL. Experience with Oracle 10g , 11g.
  • Must be proficient in creating SQL Queries, PL/SQL scripts, PL/SQL packages, functions, stored procedures, triggers, and cursors.

.NET Developer

  • Must have hands-on experience in developing software using .NET framework and knowledgeable in HTML, C# or JavaScript.
  • Responsible for developing .NET applications, anything from back-end services (windows and web services) to front-end applications.

Stimulsoft Developer

  • Must have hands-on experience in developing reports/templates using Stimulsoft and knowledgeable in .NET framework.
  • Generate technical documentation and procedures based on developed application.

Software Quality Analyst

  • Must have hands-on experience as a software tester and proficient in SQL and writing test cases.
  • Understand the functionalities of applications/web systems and work closely with analysts and developers.
  • Document results of tests executed during the duration of the whole testing phase.
